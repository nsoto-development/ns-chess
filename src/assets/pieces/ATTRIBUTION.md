# Chess piece graphics

Burnett](https://en.wikipedia.org/wiki/User:Cburnett), sourced via [lichess-org/lila](https://github.com/lichess-org/lila/tree/master/public/piece/cburnett) (Wikimedia Commons category: [SVG chess pieces](https://commons.wikimedia.org/wiki/Category:SVG_chess_pieces)).
